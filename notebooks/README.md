# Notebooks Directory

This directory contains all Jupyter notebooks documenting the analytical journey of the Bluestock MF Capstone Project.

## Notebook Sequence

| # | Notebook | Description |
|---|----------|-------------|
| 01 | `01_data_ingestion.ipynb` | Documents the raw data ingestion pipeline and initial data exploration. |
| 02 | `02_data_cleaning.ipynb` | Documents the cleaning, normalization, and validation logic applied to raw datasets. |
| 03 | `03_eda_analysis.ipynb` | Exploratory Data Analysis with 10+ interactive Plotly and Seaborn visualizations covering AUM trends, SIP flows, transaction demographics, and fund category distributions. |
| 04 | `04_performance_analytics.ipynb` | Quantitative financial modeling: CAGR, Sharpe, Sortino, Alpha, Beta, Maximum Drawdown, and composite fund scorecard. Includes benchmark comparison charts. |
| 05 | `05_advanced_analytics.ipynb` | Advanced analytics: VaR/CVaR, Rolling Sharpe, Investor Cohort Analysis, SIP Continuity, HHI Concentration, Fund Recommender. Bonus: Monte Carlo Simulation and Markowitz Efficient Frontier. |

## How to Run

```bash
# Launch Jupyter Lab
jupyter lab notebooks/

# Or open a specific notebook
jupyter notebook notebooks/03_eda_analysis.ipynb
```
