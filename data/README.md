# Data Directory

This directory contains all raw, processed, and database files for the Bluestock MF Analytics project.

> **Note:** The SQLite database file (`bluestock_mf.db`) is excluded from Git via `.gitignore` to prevent repository bloat. Run the ETL pipeline to regenerate it.

## Structure

```
data/
├── raw/          # Original CSV files ingested during Step 1 (etl_pipeline.py)
├── processed/    # Cleaned, standardized CSVs ready for database loading
└── db/           # SQLite database (bluestock_mf.db) - generated by db_load.py
```

## Regenerating the Database

```bash
python scripts/etl_pipeline.py   # Ingest raw CSVs
python scripts/clean_data.py     # Clean and normalize
python scripts/db_load.py        # Build Star Schema
```
